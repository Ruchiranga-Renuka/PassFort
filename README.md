# 🔐 PassFort

A simple yet powerful password strength checker built with **Python** and **Tkinter**.
No external packages required — just pure Python.

---

## ✨ Features

- 🔴🟠🟡🟢💪 Live strength rating as you type
- ✅ Rule-by-rule checklist breakdown
- 📊 Visual strength progress bar
- 👁️ Show / Hide password toggle
- 100% Python — no pip installs needed

---

## 📋 Requirements

- Python 3.x

That's it. Tkinter comes built into Python.

---

## 🚀 Getting Started


Run the app
```bash
python password_checker.py
```

---

## 🛡️ Password Rules Checked

| Rule | Description |
|------|-------------|
| Length | At least 8 characters |
| Uppercase | Contains at least one A–Z |
| Lowercase | Contains at least one a–z |
| Number | Contains at least one 0–9 |
| Special Character | Contains `!@#$%^&*` etc. |

---

## 📊 Strength Ratings

| Score | Rating |
|-------|--------|
| 5 / 5 | 💪 Very Strong |
| 4 / 5 | 🟢 Strong |
| 3 / 5 | 🟡 Moderate |
| 2 / 5 | 🟠 Weak |
| 0–1 / 5 | 🔴 Very Weak |
